# recuit_advisor_front

KT 채용공고 기반 AI 멘토 어드바이저 프론트엔드

## 페이지 구성

### `/` 채용공고 목록
- KT 채용공고 카드 목록 표시
- 신입/경력/인턴 필터링
- 카드 클릭 시 멘토 선택 페이지로 이동

### `/jobs/[jobId]` 멘토 선택
- 선택한 채용공고 정보 표시
- 3명의 멘토 카드 가로 배치 (김선배 / 박멘토 / 이프런티어)
- 멘토 선택 후 채팅 시작

### `/chat` 채팅
- 선택한 멘토와 1:1 채팅 인터페이스
- 채용공고 기반 맞춤 조언 (백엔드 API 연결 예정)

## 기술 스택

- Next.js 14 (Pages Router)
- TypeScript
- Tailwind CSS
