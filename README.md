# KT 채용 어드바이저 — 프론트엔드

KT 위즈 마스코트 캐릭터 **빅(Vic)** 과 **또리(Ddory)** 가 KT 취준생의 채용 준비를 도와주는 AI 멘토 서비스입니다.

## 주요 기능

- **채용공고 탐색** — KT 직무별 채용공고 카드 목록 및 필터링
- **멘토 선택** — 준비 단계에 맞는 캐릭터 멘토 선택 (빅 / 또리)
- **AI 채팅** — 채용공고 기반 맞춤 질답, 자주 묻는 질문 칩, 후속 질문 추천
- **AI 가드레일** — KT 채용 무관 질문 및 유해 질문 자동 차단
- **만족도 조사** — 상담 종료 후 설문 + KT 퇴근송(소확행) 재생

## 페이지 구성

| 경로 | 설명 |
|------|------|
| `/` | 채용공고 목록 (신입/경력 필터) |
| `/jobs/[jobId]` | 멘토 선택 (빅 / 또리) |
| `/chat?jobId=...&mentorId=...` | 1:1 AI 채팅 |

## 멘토 캐릭터

| 캐릭터 | 대상 | 스타일 |
|--------|------|--------|
| 빅 (Vic) | 구체적으로 준비 중인 취준생·경력직 | 직접적·실전형 |
| 또리 (Ddory) | KT를 탐색 중인 대학생·취준 초기 | 친근·탐색형 |

## 지원 직무

Cloud 이행 / B2B 영업 / SI 사업 제안이행 PM / SI 사업 품질관리 / 데이터 & 통화 컨설팅 / 데이터센터 사업개발

## 기술 스택

- **Next.js 14** (Pages Router)
- **TypeScript**
- **Tailwind CSS**
